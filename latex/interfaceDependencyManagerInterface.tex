\subsection{Detailed Description}
Anyone can register injectors with this class. Injectors include an interface that must be matched, a callback from that interface, one or more object keys and their matching object types.

Once registered, any object passing through the \hyperlink{interfaceDependencyManagerInterface_aeff0122054e837d487325eb6d03c5631}{set\-Dependencies()} method that has the appropriate interface will have the callback method called with the appropriate objects if they are defined.

Dependencies can be set either using the static pattern (defined in Dependency\-Manager()) or they can use D\-I themselves by requesting the \hyperlink{classDependencyManager}{Dependency\-Manager} itself as a \hyperlink{classDependencyManager}{Dependency\-Manager} (implement \hyperlink{interfaceDependencyManagerDependency}{Dependency\-Manager\-Dependency}).

To leverage D\-I in your own classes, you must make sure they are instatiated in a way that calls the dependency injector.
