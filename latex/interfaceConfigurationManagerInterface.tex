\subsection{Detailed Description}
\hyperlink{classConfiguration}{Configuration} can be provided by Y\-A\-M\-L files or I\-N\-I files, as well as arrays of settings.

\hyperlink{classConfiguration}{Configuration} can be provided as default configuration or just normal configuration. Normal configuration overrides default configuration where possible.

In most cases, overridding a configuration value replaces the existing value. However, if an array is present, the values are extended (numeric keys are appended at the end, which changes the keys, associative keys replace/extend their values as appropriate).
